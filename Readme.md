# Spotify Profile
A serverless web app to summarize your spotify profile

# Technologies
- Vercel
- Create React App (Typescript)
- Tailwind

# Setup
1. Install vercel cli `npm i -g vercel`
2. Register Spotify App 
2. Add `http://localhost:3000/callback` as redirect uri
3. set up `.env` file
4. run `vercel dev`
